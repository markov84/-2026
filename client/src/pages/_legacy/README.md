# Legacy Pages

Тук са преместени неактивните страници от `client/src/pages`.

Причина:
- да не пречат на поддръжката
- да няма объркване кой файл е реално активен
- да се пази историята без да се държи в основната папка с активните екрани

Активните страници остават в `client/src/pages`, а архивните варианти са тук само за справка.
